TSVFilterer.

The problem I am trying to solve here is to filter TSV files on Windows.

First, about TSV files: https://en.wikipedia.org/wiki/Tab-separated_values

Next, about filtering. The data in TSV files can be filtered in two ways in order to remove unnecessary data for whatever analysis we are doing:

  1. Certain columns can be removed from the file
  2. Certain rows can be removed from the file

I assume that every column contains data of the same type and every row contains data of different types.

So, columns are filtered by column number, while rows are filtered by their content.

Finally, why this tool when there is grep and its infinite filtering capabilities? Because this tool is visual, simple to use, and requires no memorization of commands. Think of it as a dumbed-down version of grep for solving a specific problem. Similarly, why this tool when Excel offers good filtering capabilities? Because this tool can handle file sizes that Excel can't.

Like all other software, this is a work in progress.
